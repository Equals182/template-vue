# template-vue

A vue-cli template.

## Usage

### vue-cli

```bash
vue init egoist/template-vue try-vue
```

### sao

Using [SAO](https://github.com/egoist/sao).

```bash
# install sao first
yarn global add sao

# from repo
sao egoist/template-vue try-vue
# or from npm
sao vue try-vue --install
```
